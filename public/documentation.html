ItFlix

API Documentation

This web application provides access to information about movies of the Golden Age of Italian Cinema (1960-1980). Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Available Endpoints

Description HTTP method Request URL Request Body Data Format Response Body Data Format
Get a list of all movies GET /movies None A JSON array of objects holding data of all movies.
Get a movie by title GET /movies/[title] None A JSON object holding data of single movie. Es. 
  {
    "title": "A Fistful of Dollars",
    "director": {
      "name": "Sergio Leone",
      "born": "1929-01-03",
      "died": "1989-04-30",
      "bio": "..."
    },
    "stars": [
      {
        "name": "Clint Eastwood",
        "born": "1930-05-31",
        "bio": "..."
      },
      {
        ...
      }
    ],
    "releaseYear": "1964"
  }
Get a list of movies by director GET /movies/directors/[directorName] None A JSON array of objects representing movies from the same director.
Get a director data by name GET /directors/[directorName] None A JSON object holding data of a specific director. Es. 
  {
    "name": "Sergio Leone",
    "born": 1929,
    "died": 1989,
    "bio": "The father of the Spaghetti Western."
  }
Get a list of movies by genre GET /movies/[genreName] None A JSON array of objects representing movies with the same genre.
Get a genre by name GET /genres/[genreName] None A JSON object holding data of a specific genre. Es. 
  {
    "name": "Spaghetti Western",
    "description": "Broad subgenre of Western..."
  }
Register a new user POST /users A JSON object holding a user's info. Es. 
  {
    "username": "Rossi",
    "email": "rossi@xyz.com",
    "password": "1234",
    "birthday": "1972-03-08"
  }
A JSON object holding data of the new user. Es. 
  {
    "username": "Rossi",
    "email": "rossi@xyz.com"
    "password": "1234",
    "birthday": "1972-03-08T00:00:00.000Z",
    "_id": "63f52fb800370a3552ad9eb0",
    "favouriteMovies": [],
    ...
  }
Update user info (username, email, password, birthday) PUT /users/[username] A JSON object holding the updated info. Es. 
  {
    "username": "MarioRossi",
    "email": "mario.rossi@xyz.com"
  }
A JSON object holding data of the updated user. Es. 
  {
    "username": "MarioRossi",
    "email": "mario.rossi@xyz.com"
    "password": "1234",
    "birthday": "1972-03-08T00:00:00.000Z",
    "_id": "63f52fb800370a3552ad9eb0",
    "favouriteMovies": [],
    ...
  }
Add movie to user's list of favorites PUT /users/[username]/movies/[movieId] None A JSON object holding data of the updated user. Es. 
  {
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com",
    "password": "1234",
    "birthday": "1972-03-08T00:00:00.000Z",
    "_id": "63f52fb800370a3552ad9eb0",
    "favouriteMovies": [
      "63ed66e38e058a92de1d455c"
    ]
  }
Remove movie from user's list of favorites DELETE /users/[username]/movies/[movieId] None A JSON object holding data of the updated user. Es. 
  {
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com",
    "password": "1234",
    "birthday": "1972-03-08T00:00:00.000Z",
    "_id": "63f52fb800370a3552ad9eb0",
    "favouriteMovies": []
  }
Delete user by username DELETE /users/[username] None A JSON object holding data of the deleted user.
