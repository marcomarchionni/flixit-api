Spaghetti Western API

Documentation

This web application provides users with access to information about the top Spaghetti Western movies. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Available Endpoints

Description HTTP method Request URL Request Body Data Format Response Body Data Format
Get a list of all movies GET /movies None A JSON object holding data of all movies.
Get a movie data by title GET /movies/[title] None A JSON object holding data of single movie. Es. 
  {
    "title": "My Name Is Nobody",
    "director": "Tonino Valerii",
    "writers":[
      "Sergio Leone",
      "Fulvio Morsella",
      "Ernesto Gastaldi"
      ],
    "stars": [
      "Terence Hill",
      "Henry Fonda",
      "Jean Martin"
      ],
    "year": 1973,
  }
Get a list of movies by director GET /movies/directors/[director] None A JSON object holding data of all movies of a specific director.
Get a director data by name GET /directors/[name] None A JSON object holding data of a specific director. Es. 
  {
    "name": "Sergio Leone",
    "born": 1929,
    "died": 1989,
    "bio": "The father of the Spaghetti Western."
  }
Register a new user POST /users A JSON object holding a user's info. Es. 
  {
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com"
  }
A JSON object holding data of the new user. Es. 
  {
    "id": 1,
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com"
  }
Update a user name PUT /users/[id]/[name] None A JSON object holding data of the updated user. Es. 
  {
    "id": 1,
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com"
  }
Add movie to user's list of favorites PUT /users/[id]/movies/[movieId] None A JSON object holding data of the updated user. Es. 
  {
    "id": 1,
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com"
    "movies": [
      {
        "title": "My Name Is Nobody",
        "director": "Tonino Valerii",
        "writers":[
          "Sergio Leone",
          "Fulvio Morsella",
          "Ernesto Gastaldi"
        ],
        "stars": [
          "Terence Hill",
          "Henry Fonda",
          "Jean Martin"
          ],
        "year": 1973,
      }
    ]
  }
Remove movie from user's list of favorites DELETE /users/[id]/movies/[movieId] None A JSON object holding data of the updated user. Es. 
  {
    "id": 1,
    "name": "Mario Rossi",
    "email": "mario.rossi@xyz.com"
    "movies": []
  }
Delete user by email DELETE /users/[email] None A text response confirming that the user with email [email] has been removed.
